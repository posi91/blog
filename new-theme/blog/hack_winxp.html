Vulnerando Windows XP en 5 minutos - La importancia de tus datos y migración a una distribución linux.

Miercoles 19 de Febrero @po51


Introducción

Este es un video con fines educativos. Se trata de una vulnerabilidad para la ejecución remota de código. Un atacante que aprovechara esta vulnerabilidad podría tomar el control completo de un sistema afectado de forma remota. En los sistemas basados en Microsoft Windows 2000, Windows XP y Windows Server 2003, un atacante podría aprovechar esta vulnerabilidad sobre RPC sin autenticación y ejecutar código arbitrario. Si falla el intento de aprovecharse de esta vulnerabilidad, también podría dar lugar a un bloqueo en Svchost.exe. Si se produce dicho bloqueo, afectará al servicio del servidor. El servicio del servidor proporciona compatibilidad con el uso compartido de archivos, impresoras y canalizaciones con nombre a través de la red. La causa de esta vulnerabilidad es el servicio del servidor, que no controla correctamente las solicitudes de RPC especialmente elaboradas para ello.

Didulación

Se muestra la importancia de que este año Microsoft a anunciado que dejara de dar soporte a Windows XP, a raíz de esto como ya muchos sabemos existen diversidad de fallos de seguridad sobre esta plataforma que podríamos explotar fácilmente con metasploit. Te importa tu información ? Si tienes windows xp te aconsejaron que migraras a Ubuntu o mejor aun a Debian o a otra distribución linux, ya sea por que tu maquina es vieja, es anticuada en hardware y no podes migrar a win 7, una solución seria el software libre, igual que muchos fui "Windolero" a raiz del tiempo de mi manera de pensar de ver las cosas, me pase a linux comencé con Ubuntu recuerdo tanto, después me pase a Linux Mint, también tuve de sistema base backtrack pero no te lo recomiendo a caso de que te dediques al pentesting, por ultimo conocí debian tengo que admitirlo es la distribución mas estable, y mas cuando un bruto como yo tiene debían rompemos paquetes en fin y aun así sigue estable, rápido, y con todo lo que necesito. Accesorios, gráficos, Internet, juegos(steam), programación, multimedia, libre office, herramientas de diseño, herramientas para desarrollo web, herramientas para consultoria de seguridad informática, en fin tengo una gran cantidad de herramientas desde synaptic, puedo añadir repositorios para instalar cualquier otra herramienta de otra distribución linux. Bueno esto solo era un mensaje y un ejmplo de lo que podira pasarle a tu preciosa maquina con windows xp, decidete el software libre es una buena desicion.
